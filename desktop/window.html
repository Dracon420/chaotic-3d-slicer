① Trust the certificate (one time)
  1. On the phone (same Wi‑Fi), scan the QR / open the link to download the certificate.
  2. Android: install it as a CA certificate (Settings → Encryption & credentials → Install a certificate → CA certificate) — not “Wi‑Fi certificate”. Only the CA slot makes the browser trust it and show Install. iPhone: Settings → Profile Downloaded → Install, then Settings → General → About → Certificate Trust → enable it.
  3. If the link won't open, use instead (accept the warning once).

Each PC generates its own certificate — if you run the app on a second PC, install that PC's certificate too (the phone can hold several).

② Open & install the app
